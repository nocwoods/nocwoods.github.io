bibex with LaTex

  • steps from scratch:

    1. setup .bib file for all bibliography;

    2. define bibliography-style shown in final PDF file;

    3. follow below template to build .tex file;

    4. insert citation at place you needed

    5. generate and print out PDF file with additional option:

      latex mytex # generate document with question mark on unknown reference;
bibtex mytex # link .bib and generate reference information;
latex mytex # generate all correct citation;
latex mytex # make sure everything is clean and smoothly;
dvipdf mytex.dvi # PDF print out;

  • step1: .bib file

    • search in google scholar or others

    • https://scholar.google.com

    • http://xueshu.baidu.com/

    • https://cn.bing.com/academic

    • and export as BibTex format,examples:

      @article{glass1992the,
title={The turn model for adaptive routing},
author={Glass, Christopher J and Ni, Lionel M},
volume={20},
number={2},
pages={278--287},
year={1992}}

title={A survey of wormhole routing techniques in direct networks},

  • step2: template and format saved in .bst file; different article/conference could be different;

bibtex in emacs-org (with org-ref)

  • install org-ref from MELPA

    Alt-x list-packages
org-ref

  • setup default .bib file for search:

    Alt-x customization-variable
org-ref-default-bibliography

  • update latex export with additional option:

    '(TeX-command-extra-options "--shell-escape --bibtex ")

  • update .org template like:

    #+LATEX_HEADER: \usepackage{cite}
....
insert citation here with "C-c ]"
....
bibliography:bibtex.bib
bibliographystyle:unsrt

  • insert citation

    ""C-c ]" -> ""C-n/p" -> "TAB" -> "f2""

  • also support html export;(without number)

  • reference

    • https://github.com/jkitchin/org-ref
    • http://kitchingroup.cheme.cmu.edu/blog/2014/05/13/Using-org-ref-for-citations-and-references/
