github-pages for blogs

github pages for blogger deployed server, based on hugo.

github-blog-hugo

  • github account apply and initialization

    • apply account; and create one initial repo;
    mkdir /var/vobs/blogger/blog
echo "# nocwoods.github.io" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/nocwoods/nocwoods.github.io.git
git push -u origin master

  • clone github pages and test with hugo

    git clone https://github.com/nocwoods/nocwoods.github.io.git
cd blogger ; cp -r ../base/* . ; rm -rf public/
git submodule add -b master https://github.com/nocwoods/nocwoods.github.io.git public
hugo
cd public/
git add -A .
git commit -m "1st version"
git push origin master

  • preprocess

    • update config.toml file for correct base_url

  • postprocess

    • in order to automatically git-push, it worth to use a shell to run the repeated git-push;
    • in order to avoid github push with account/password input everytime, it worth to enable SSH-Key or GPG-Key for auth;
