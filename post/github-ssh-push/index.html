github accessing with SSH

github accessing with SSH, and upload public-key.

github-ssh-push

  • steps for ssh access github:

    • ssh-keygen generate ssh key-pair;

    • add private key into ssh-agent;

    • upload public key to github;

    • clone github based on ssh protocol:

      git@github.com:nocwoods/nocwoods.github.io.git
